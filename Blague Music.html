Blague Music: Your Personalized Soundtrack

challenge：designing and optimizing a music web page

Date: June - July 2023

Skills: Website Design, User Experience, User Interface Design

Tools: Figma, Adobe Illustrator

Check The Figma

Design Process:

My goal for this project was to create a music web page called Blague. I conducted market research, analyzed other music websites, and gathered user feedback to inform the initial design and prototype. Blague's features were refined through iterative testing, and I played a role in the entire product design process, from concept to implementation, to align with our goal. After the launch, we continuously updated the website based on user feedback to keep it fresh and relevant.

User Experience Improvements：

Navigation and Organizational Structure：

· Issue: Unclear navigation and disorganized content structure.

· Fix: Implemented a hierarchical structure with clear visual hierarchy, including drop-down menus for streamlined navigation. Conducted card sorting exercises to realign content with user expectations.

Interaction Design and User Interface：

· Concern: Non-intuitive design of buttons and icons.

· Fix: Redesigned buttons and icons for distinctiveness and descriptiveness. Adjusted interactive elements for improved visibility through shadows, borders, and color changes. Simplified icons to better represent their functions.

Lack of Vibrancy in User Interface：

· Problem: The interface lacks visual vibrancy.

· Suggestion: Introduced color contrasts, animations, and dynamic elements like hover effects. Utilized a varied color palette that aligns with the brand identity for enhanced visual appeal.

Design Process and Learning：

· Typography and Layout: Explored the use of various fonts and sizes in Figma to direct attention and maintain aesthetics. Improved information delivery through conscious layout choices.

· Animation Techniques: Investigated animation techniques in Figma to enliven web pages and enrich user interaction.

· Inspiration and Creativity: Sought inspiration from existing web designs, analyzing current trends and user preferences. Infused personal creativity into designs to create unique and resonant interfaces.
